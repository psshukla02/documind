DocuMind combines RAG, synthetic data generation, and structured prompt engineering into a single web app for developers. Ingest any documentation site, ask grounded technical questions, and generate polished Markdown docs — with inline citations and live confidence scoring.

Features

📚 RAG pipeline

Scrape → chunk → embed → store (FAISS) → retrieve → generate. Sources cited inline.

💬 Chat with citations

Every answer carries [S#] markers that link to the source document — no hidden claims.

📝 Documentation generator

Produce structured Markdown docs from a topic or a code snippet, optionally grounded in your KB.

🧪 Synthetic data

Generate diverse Q&A pairs (factual, reasoning, edge-case, example) with a strict JSON contract.

🧠 Prompt engineering

Role separation, instruction hierarchy, edge-case handling, explicit output contracts.

📈 Live metrics

Latency, retrieval score, token usage, and a rolling event log — refreshed every 5 seconds.

Demo

Run it locally in two commands. Vite dev server proxies to FastAPI, so the frontend makes same-origin requests. No API key? DocuMind ships a stub mode so every UI flow still works offline.

Try ingesting https://fastapi.tiangolo.com/, then ask "What is FastAPI and how does it use Pydantic?" — you'll see the answer, the citation pills, and a confidence bar in the UI. 

# backend
cd backend
cp .env.example .env
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000

# frontend (in another terminal)
cd frontend
npm install
npm run dev
# → http://localhost:5173
